# CRE8 Curriculum Narrative Outlines

This folder contains episode-outline artifacts for a scripted educational programming series built on top of the CRE8 90-day plan.

## Primary educational goals
1. Teach programming in an accessible, beginner-friendly way.
2. Show the "How did we build CRE8?" process in digestible daily steps.
3. Help CRE8 users learn the platform and eventually build their own application on top of CRE8, with **XtraType** as the first full handoff example.

## Current curriculum artifacts
- `M1_GOVERNANCE_RUNTIME_FOUNDATIONS_OUTLINE.md` — Days 1–21 narrative outlines plus M1→M2 entry-gate episodes.
- `M2_SCHEMA_AUTH_POLICY_CORE_OUTLINE.md` — Days 22–45 narrative outlines plus M2→M3 entry-gate episodes.

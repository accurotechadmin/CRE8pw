# Documentation Roadmap & Backlog

_Last updated (UTC): 2026-04-05_

This backlog tracks future refinements for an already authoritative doc set.

## Near-term improvements

- Add architecture diagrams (context, component, request-sequence).
- Add endpoint-by-endpoint request/response examples for all routes.
- Add expanded operations playbooks for key rotation and incident classes.
- Add generated file-cross-reference index for docs-to-code traceability.

## Ongoing maintenance cadence

- Update docs in every PR that changes route behavior, policies, or data contracts.
- Reconcile docs/dev execution artifacts into top-level references when behavior becomes permanent.
- Re-verify endpoint and middleware claims against tests each release cycle.

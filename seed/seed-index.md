# CRE8 Seed Master Index

## Introduction
These seed documents form the foundational source set used to create the rest of the canonical SSOT guiding documents for the Credential Registry Engine (CRE8) software application platform. Together they define the baseline architecture, governance constraints, security posture, delegation model, parity obligations, and documentation maturation path.

## Document Index
- `seed-intro.md` — Root seed canon overview describing CRE8 identity, non-negotiable architecture shifts, and obligations for future SSOT authoring.
- `CRE8_KEYPAIR_MODEL_BASE_INVENTORY.md` — Authoritative architectural inventory for the ID-keypair-first model, actors, authority chain, and acceptance baseline.
- `CRE8_SEED_CANON_INDEX.md` — Canon navigation map defining reading order, domain ownership, terminology constraints, and maturation expectations.
- `CRE8_PERMISSION_AND_DELEGATION_SEED.md` — Hierarchical delegation and bounded permissioning rules anchored to PDP-first enforcement.
- `CRE8_KEY_LIFECYCLE_AND_CRYPTOGRAPHY_SEED.md` — Key lifecycle, cryptographic proof, revocation, and immutable event requirements.
- `CRE8_SURFACES_AND_CLIENT_PARITY_SEED.md` — Operational surface model and UI/API parity expectations across first-party and third-party clients.
- `CRE8_CONTENT_AUDIENCE_AND_FEED_SEED.md` — Content targeting, audience modeling, interaction controls, and deterministic feed behavior requirements.
- `CRE8_API_CONTRACT_AND_ERROR_SEED.md` — API envelope invariants and deterministic deny/error mapping obligations.
- `CRE8_EXTENSIBILITY_AND_MODULE_PATTERN_SEED.md` — Developer-first extension patterns and module boundaries that preserve core invariants.
- `CRE8_SEED_PRESERVATION_MATRIX.md` — Preservation matrix tracking preserved truths, mandatory redesigns, and intentionally dropped assumptions.
- `CRE8_SEED_CANON_ASSESSMENT_REPORT.md` — Gap analysis and build-forward blueprint for maturing the seed canon into production-grade SSOT.
- `CRE8_REPO_STUDY_REPORT.md` — Repository-level synthesis describing document relationships, current maturity, and implementation implications.
- `seed-index.md` — This master seed index and orientation guide for the `/seed` document set.

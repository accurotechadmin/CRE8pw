# CRE8 New SSOT Seed Set (`/newdev`)

This directory is the staging SSOT seed for the upcoming clean-slate CRE8 repository. It intentionally preserves the strongest architectural and governance traits from the current document corpus while re-centering the platform on mandatory ID-keypair minting for all delegated key classes.

The initial documents are concise by design. Each file provides an authoritative snapshot of intent and scope so teams can begin implementation planning immediately, then mature each document into complete production-grade SSOT references without losing conceptual continuity.

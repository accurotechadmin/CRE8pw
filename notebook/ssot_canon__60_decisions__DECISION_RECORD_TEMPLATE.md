# Decision Record Template

_Status: adopted_
_Last updated (UTC): 2026-04-08_

## Required fields
- Decision ID
- Date
- Status (`proposed|accepted|superseded`)
- Context/problem statement
- Decision
- Alternatives considered
- Consequences (positive/negative)
- Verification and rollout impacts
- Related SSOT docs

## Quality bar
A decision record is invalid if it lacks either explicit consequences or verification implications.

# CRE8 Development Workspace

This `/dev` tree is the developer-facing workspace for translating the CRE8 SSOT canon into a runnable production codebase.

## Purpose
- Provide implementation-focused onboarding material.
- Document codebase scaffolding standards and build workflow expectations.
- Bridge SSOT requirements to concrete development execution steps.

## Authority model
1. `docs/ssot_canon/` is the normative source of truth.
2. `/dev/docs/` explains how to implement that truth safely in code.
3. If `/dev/docs/` conflicts with SSOT canon, SSOT canon wins.
